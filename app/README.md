# app/ — SESI Processo Seletivo (frontend)

Aplicação web do **Sistema de RH / Gestão de Vagas** do SESI. SPA em **Vite + React 19 + TanStack
Router/Query/Table + shadcn/ui + Tailwind v4**.

> Este é o pacote do app dentro do repositório. A visão do produto, o vocabulário e a documentação completa
> estão na **raiz**: [`../README.md`](../README.md) · [`../CONTEXT.md`](../CONTEXT.md) ·
> [`../docs/`](../docs/README.md).

## Rodar

```bash
pnpm install
pnpm dev
```

Comandos: `pnpm build` · `pnpm lint` · `pnpm format` · `pnpm test`. Guia completo em
[`../CONTRIBUTING.md`](../CONTRIBUTING.md). Referência da stack em
[`../docs/engineering/stack.md`](../docs/engineering/stack.md).
