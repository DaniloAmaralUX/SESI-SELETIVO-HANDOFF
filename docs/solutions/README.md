Engineering: cada problema resolvido de forma não-óbvia vira um documento pesquisável aqui, e o próximo
`/ce-brainstorm` / `/ce-plan` lê estes arquivos como grounding.

que se resolve um problema custa pesquisa; documentado, a próxima ocorrência custa minutos.
