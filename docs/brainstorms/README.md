# docs/brainstorms

Saídas de **`/ce-brainstorm`** — exploração colaborativa do **o quê** antes de planejar.

Cada arquivo é um *requirements-only unified plan* (PRD leve): escopo, comportamento visível, critérios de
sucesso e não-objetivos de uma feature — **sem** detalhe de implementação. É o insumo que `/ce-plan` enriquece
para virar um plano implementation-ready em [`../plans/`](../plans/).

> Convenção: `AAAA-MM-DD-<slug>.md`. Aterrado em [`../../CONTEXT.md`](../../CONTEXT.md),
> [`../product/`](../product/) e [`../../STRATEGY.md`](../../STRATEGY.md).
